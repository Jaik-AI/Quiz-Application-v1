# Quiz-Application-v1
A Python-based quiz application that evaluates user responses, calculates scores, percentages, and displays pass/fail results.

## Features

- Multiple-choice questions
- Input validation (A, B, C, D only)
- Score calculation
- Percentage calculation
- Pass/Fail result generation
- Displays correct answer when the user is wrong

## Technologies Used

- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Input Validation

## How to Run

```bash
python main.py
```
